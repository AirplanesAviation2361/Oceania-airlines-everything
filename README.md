# Oceania-airlines
# ✈️ Oceanic Express Airways (OEA)

**ICAO:** OEA  
**Hub:** Auckland International Airport (NZAA)  
**Fleet:** Boeing 787 only  
**Platform:** GeoFS  

## About
Oceanic Express Airways (OEA) is a long-haul virtual airline operating an all-Boeing 787 fleet.  
OEA is the founder of the **OEA Alliance**, connecting major international airlines.

## Liveries
- **OEA Normal (Air France–style example)**
- **OEA Alliance**
- **OEA Trans**

## OEA Alliance Members
- Air France  
- British Airways  
- Delta Air Lines  
- Spirit Airlines  
- American Airlines  
- Emirates  

## Installation (GeoFS)
1. Upload this repository to GitHub
2. Copy the RAW link of `airline.json`
3. Open GeoFS → LiverySelector
4. Tap **+ Add Airline**
5. Paste the link
6. Save & reload aircraft

## Notes
- All liveries are for **Boeing 787**
- Alliance livery represents all partner airlines
